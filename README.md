# base-files-email
Responsive email base files

A Responsive email template in which you can easily drop you get and header image in.  3 x buttons are on the right hand side for links to other sites.


##Testing in

+ Outlook 2003
+ Outlook 2007
+ Outlook 2010
+ Outlook 2013
+ Thunderbird
+ Android 2.3
+ Android 4.0
+ Gmail (Android)
+ IPad
+ IPhone 5s
